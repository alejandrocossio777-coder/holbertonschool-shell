# holbertonschool-shell

Repositorio de Holberton School con proyectos de shell en Bash (Ubuntu 22.04 LTS):

- **basics** — Navegación, listados, archivos y directorios.
- **permissions** — chmod, propietario, grupos, usuarios (su, whoami, chown, etc.).

Cada carpeta de proyecto incluye su propio `README.md` con la descripción de cada script.
